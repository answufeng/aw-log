# aw-log Demo 功能矩阵

| 区域 | 能力 |
|------|------|
| 开关 | Debug / 文件 / 崩溃 / 脱敏 / 紧凑格式 |
| 级别 | 最低级别 Chips + 发送级别 |
| 发送 | 文本 / 异常 / JSON / XML / 脱敏样例 |
| 文件 | 信息、压缩、导出 ZIP、清理 |
| 其它 | 并发写、**崩溃**（会杀进程，仅调试用） |

与 README **生产环境检查清单** 对照，Release 勿照搬 demo 全开。工具栏 **「演示清单」** 可查看本摘要。

## 推荐手测（边界与极端场景）

| 场景 | 建议操作 |
|------|----------|
| 磁盘 | 日志目录空间耗尽或只读挂载时的行为 |
| 并发 | 多线程狂打日志，文件树是否丢日志或阻塞 |
| 脱敏 | 自定义规则下敏感字段是否仍可能泄露（正则边界） |
| 进程 | 崩溃演示仅调试包使用；与第三方崩溃 SDK 同进程时确认 handler 不重复安装 |
